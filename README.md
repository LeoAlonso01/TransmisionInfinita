## 📖 Sobre este proyecto

*Transmisión Infinita* es un podcast humanista que explora preguntas fundamentales del ser humano:  
¿Cómo vivir con ética en tiempos de fragmentación?  
¿Qué puede enseñarnos el pensamiento simbólico antiguo sobre la ansiedad moderna?  
¿Es posible la fraternidad sin ideología?

No somos una institución ni representamos doctrina alguna. Abordamos el hermetismo, la filosofía y la espiritualidad como **símbolos para reflexionar**, no como verdades absolutas. Priorizamos siempre la **salud mental, la dignidad humana y el pensamiento crítico**.

Este repositorio contiene el código fuente de la [página web oficial](https://transmisioninfinita.com), construida con HTML, CSS y JavaScript puro.

## 🛠️ Tecnologías usadas

- **HTML5** – Estructura semántica
- **CSS3** – Diseño responsivo, animaciones suaves, tipografía humanista
- **JavaScript vanilla** – Sin frameworks; solo lógica ligera para UX
- **Google Fonts** – *Cormorant Garamond* (serif) + *Inter* (sans)
- **Hosting** – Desplegado en [Vercel](https://vercel.com) / [Netlify]
